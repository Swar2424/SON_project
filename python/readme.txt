Le fichier Main.py doit être exécuté dans un environnement python avec les dépendances suivantes :

-Tkinter :			(install tk)
-CustomTkinter :		(install customtkinter)
-Pillow :			(install Pillow)
-PyDub : 			(install pydub)
-PyAudio :			(install PyAudio)

--------------------------------------------------------------------------------------------------------------------

Les enregistrements sont générés sous le nom de "Musique.wav". Tout nouvel enregistrement écrase le précédent.

--------------------------------------------------------------------------------------------------------------------

En cas de problème rencontré, redémarrer l'application. Si le problème persiste, redémarrer le kernel.

--------------------------------------------------------------------------------------------------------------------

Le programme n'a pas pu être testé sur Linux.

--------------------------------------------------------------------------------------------------------------------

Si de trop nombreux sons sont activés en même temps, la lecture audio peut être ralentie lors de l'exécution du
programme. Cependant, la vocation principale de ce programme est la création de fichiers audios, et ceux-ci ne 
présentent pas de ralentissement ou de problèmes de lecture, peu importe le nombre de sons joués.
